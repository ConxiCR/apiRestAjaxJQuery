# Api Rest #

## Recerca de la API
1. Al buscador GOOGLE fer reserca de "api rest online".
2. Entrar en la ruta https://jsonplaceholder.typicode.com per fer el exercici de prova.
3. Entrar en el endpoint post. Al enllaç entrarem al url a on hi ha allotjat l'objecte json. Array d'objectes javascript expresada en forma de clau valor.

## Tasca: Recollir la informació i mostrar els titols en pantalla
